# VuePress 文档格式化 Skill

## 概述

这是一个专业的 VuePress 文档格式化技能，用于统一和美化 Markdown 文档格式，提升文档的可读性和专业性。

## 核心功能

### 1. Frontmatter 标准化

为每个文档添加完整的元数据：
- title：文档标题
- description：简短描述
- date：创建/更新日期
- tags：标签列表
- categories：分类

### 2. 页面头部装饰

使用统一的页面样式：
- 居中的 H1 标题
- Element Plus 分隔线
- 装饰性副标题

### 3. 内容组织结构

标准的文档结构：
- 概述
- 核心功能
- 使用说明
- 示例
- 注意事项
- 总结

### 4. VuePress 自定义容器

使用容器增强文档：
- `tip`：有用的提示
- `warning`：注意事项
- `danger`：危险警告
- `info`：补充信息
- `details`：可折叠内容

### 5. 表格美化

优化表格格式：
- 清晰的表头
- 合理的对齐
- 表格标题
- 使用 ✓ 和 ✗ 表示是/否

### 6. 代码块优化

完善代码展示：
- 语言标识
- 代码说明
- 行号和行高亮
- 注释说明

### 7. 图片优化

规范图片使用：
- Alt 文本
- 图注说明
- Emoji 类型标识
- 合适的文件名

### 8. 列表优化

使用清晰的列表：
- 图标标识（✅ ⏳）
- 分组标题
- 有序和无序列表
- 复选框列表

### 9. 链接优化

改进链接文本：
- 有意义的文本
- 避免使用 "点击这里"
- 清晰的目标描述

### 10. 文档模板

提供多种模板：
- 基础文档模板
- API 文档模板
- 教程文档模板

将 `vuepress-doc-formatter` 文件夹或压缩包放到合适的技能目录中。

技能会在以下情况下自动触发：

### 技能工作流程

1. **分析需求**：理解用户的格式化需求
2. **读取文档**：使用 `read_file` 读取现有文档
3. **应用规范**：按照 `format-specs.md` 的规范进行格式化
4. **使用模板**：参考 `assets/templates/` 中的模板
5. **保存文档**：使用 `write_to_file` 或 `replace_in_file` 保存
6. **验证质量**：使用质量检查清单验证结果

## 常见问题

### Q: 这个技能适用于哪些文档？

A: 适用于使用 VuePress 构建的项目的 Markdown 文档，特别是技术文档、API 文档、教程等。

### Q: 可以自定义格式规范吗？

A: 可以。修改 `references/format-specs.md` 中的规范，技能会按照新的规范进行格式化。

### Q: 如何添加新的文档模板？

A: 在 `assets/templates/` 目录下创建新的模板文件，并在技能说明中引用。

### Q: 可以批量处理多个文档吗？

A: 可以。技能支持批量处理，会列出目录下的所有文档并逐个格式化。

### Q: 格式化会改变原有内容吗？

A: 格式化只会调整格式，不会改变原文档的实质性内容。但如果需要删除内容，技能会先与用户确认。
