## 使用说明

### 1. Frontmatter 配置

每个文档都应该包含完整的 frontmatter：

```yaml
---
title: 文档标题
description: 简短描述文档内容
date: 2024-03-12
tags: [标签1, 标签2]
categories: [分类]
---
```

::: tip 提示
description 字段应该简明扼要，不超过 50 个字符。
:::

### 2. 页面头部装饰

使用统一的页面头部样式：

```html
<h1 style="text-align: center;">文档标题</h1>
<el-divider border-style="dashed">
    <div> ~ 副标题 ~ </div>
</el-divider>
```

### 3. 内容组织

使用清晰的章节结构：

**主要章节**：
- 概述：简要介绍文档内容
- 核心功能：列出主要功能点
- 使用说明：详细的使用步骤
- 示例：实用的代码示例
- 注意事项：重要的注意点
- 总结：文档总结

## 示例

### 代码示例

这是一个带说明的代码示例：

```cpp
// C++ 类定义示例
class MyClass {
public:
    // 构造函数
    MyClass(int value) : m_value(value) {
        // 初始化
    }

    // 获取值
    int getValue() const {
        return m_value;
    }

    // 设置值
    void setValue(int value) {
        m_value = value;
    }

private:
    int m_value;  // 成员变量
};
```

**代码说明**：
- 展示了基本的 C++ 类定义
- 包含构造函数、公有方法和私有成员
- 使用了初始化列表
- 方法都有注释说明

## 常见问题

### 问题 1：如何添加代码块高亮？

**解答**：在代码块开始标记后添加行号或行范围：

```cpp {1,3-5}
// 这行会高亮
// 这行不会高亮
// 这行会高亮
// 这行会高亮
// 这行会高亮
```

### 问题 2：如何创建可折叠的内容？

**解答**：使用 `details` 容器：

```markdown
::: details 点击展开
这里是可以折叠的内容
:::
```
