## 环境准备

### 系统要求

- 操作系统：Windows / macOS / Linux
- 软件版本：版本号或更高

### 安装步骤

#### 步骤 1：安装软件

::: details Windows 系统
```bash
# 使用 Chocolatey
choco install software-name

# 或手动下载安装
访问 https://example.com/download 下载安装包
```
:::

::: details macOS 系统
```bash
# 使用 Homebrew
brew install software-name
```
:::

::: details Linux 系统
```bash
# Ubuntu/Debian
sudo apt-get install software-name

# CentOS/RHEL
sudo yum install software-name
```
:::

#### 步骤 2：配置环境

```bash
# 配置环境变量
export SOFTWARE_HOME=/path/to/software
export PATH=$PATH:$SOFTWARE_HOME/bin
```

#### 步骤 3：验证安装

```bash
# 检查版本
software-name --version
```

**预期输出**：
```
software-name version 1.0.0
```
