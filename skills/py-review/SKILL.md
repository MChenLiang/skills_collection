# 代码审核技能

此技能提供代码审核的专业能力，专注于识别和删除未使用的函数和死代码。

## 核心工具

本技能使用 `code_cleaner.py` 脚本执行代码清理操作：
- **脚本路径**：`.codebuddy/skills/py-review/scripts/code_cleaner.py`
- **执行方式**：使用当前 Python 解释器执行
- **命令行框架**：使用 Fire 库提供命令行接口

## 何时使用此技能

在以下情况下使用此技能：
- 用户请求代码审核
- 用户要求清理未使用的函数
- 用户希望识别死代码
- 用户请求通过删除不必要的代码来优化代码
- Maya/Rez 项目构建前的代码清理

## 命令行接口

### 基本语法

```bash
<当前Python解释器> <code_cleaner.py路径> <命令> <目标目录>
```

**注意**：使用 `sys.executable` 获取当前 Python 解释器路径。

### 可用命令

| 命令 | 说明 |
|------|------|
| `preview` | 预览未使用的函数（不修改代码） |
| `clean` | 删除未使用的函数（实际修改代码） |
| `quick_preview` | 快速预览（使用默认安全模式） |
| `quick_clean` | 快速清理（使用默认安全模式和备份） |

## 使用流程

### 1. 预览阶段（必须）

先预览，确认无误后再执行清理：

```bash
<当前Python解释器> .codebuddy/skills/py-review/scripts/code_cleaner.py preview <目标目录>
```

### 2. 执行清理（用户确认后）

```bash
<当前Python解释器> .codebuddy/skills/py-review/scripts/code_cleaner.py clean <目标目录>
```

## 内置功能

脚本已内置以下智能处理：

- **自动排除**：测试文件、构建目录、第三方包等
- **Qt 信号保护**：保留 Qt 信号处理函数
- **Maya 特殊支持**：识别 Maya 插件入口函数
- **备份机制**：修改前自动备份文件
- **安全模式**：默认保守策略，避免误删

## 输出说明

- **预览模式**：显示将被删除的函数列表，不实际修改
- **清理模式**：显示删除进度和备份位置
- **备份目录**：`<项目目录>/.backup_cleaner`
